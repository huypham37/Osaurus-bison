# Resources

This folder contains app resources like images and GIF animations.

## Adding the Thinking Animation GIF

1. Place your GIF file in this folder
2. Name it: `thinking-animation.gif`
3. In Xcode, right-click on the `Resources` folder and select "Add Files to osaurus..."
4. Select your `thinking-animation.gif` file
5. Make sure "Copy items if needed" is checked
6. Make sure the target is set to "osaurus"

The GIF will then be displayed when the assistant is thinking/streaming.
